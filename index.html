#2. Tool Proposed: Vault

         
Role: Proposer
Status: Assess
Use cases/Potential uses:

 1. Vault is a tool for secrets management, encryption as a service, and privileged access management. Vault is an effective tool for managing your secret
 2. The major advantage of using Vault over ASM is it’s ability to enable customizations and writing custom authentication plugins
 3. Vault is not dependent to AWS and this application can be used across departments where they may not use or have access to AWS.

 EMIS Tech Radar link: https://radar.emisgroup.uk/technology/vault/
 Github Link: https://github.com/emisgroup/tech-radar/pull/275

Approval process:

1. Evaluate and research Hashicorp’s Vault by setting it up in local
2. Comparison between Vault and AWS Secret’s manager
3. Collaborate with security team on the plan for Thycotic

#3. Tool Proposed: Grafana

         
Role: Proposer
Status: Adopt
     
Documented for rationalising Grafana from Assess to Adopting stage in EMIS Tech radar 
   
Use cases/Potential uses:

1. Group Internal Systems have been using Grafana extensively for years to monitor it’s application server loads, API execution time, slow queries, memory usage and error
2. Grafana users can make use of a large ecosystem of ready-made dashboards for different data types and sources.
3. Internal Systems use MySQL Data Source - Native Plugin to monitor database metrics
4. Grafana provides Slack integration and ships with a built-in alerting engine that allows users to attach conditional rules to dashboard panels that result in triggered Alerts
     EMIS Tech Radar Link: https://radar.emisgroup.uk/technology/grafana/
Github Link: https://github.com/emisgroup/tech-radar/pull/275

1. Rationalise live key points and usage of Grafana in Internal systems to showcase the product strength
